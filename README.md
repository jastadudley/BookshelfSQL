# BookshelfSQL

## Overview

As a developer working toward a career in machine learning engineering, I built this program to create and manage a relational book database using Python and SQLite. The program demonstrates full CRUD operations — inserting, modifying, deleting, and retrieving data across multiple related tables. This project serves as the storage layer for The Shelf, my intelligent book recommendation engine, and builds directly on the dataset explored in Sprint 1.
